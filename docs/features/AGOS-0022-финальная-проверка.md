## Описание

Финальная интеграционная проверка всех компонентов AG-OS. Запуск полного набора тестов с coverage, проверка линтинга через ruff, верификация что все 7 фаз работают вместе корректно. Финальный коммит.

## Acceptance Criteria

- [ ] Все тесты проходят (pytest -v --tb=short --cov)
- [ ] Coverage отчёт сгенерирован
- [ ] ruff check без ошибок
- [ ] Нет пропущенных импортов или циклических зависимостей
- [ ] Финальный коммит создан

## Затрагиваемые модули

- Все модули — интеграционная проверка

## Ключевые интерфейсы

Нет новых интерфейсов — интеграционная проверка.

## Edge Cases

- Пропущенные импорты между модулями
- Циклические зависимости
- Несовместимость версий зависимостей

## План реализации

### Step 1: Запустить все тесты

```bash
pytest tests/ -v --tb=short --cov=. --cov-report=term-missing
```

Expected: All passed, coverage report

### Step 2: Проверить линтинг

```bash
pip install ruff
ruff check .
```

### Step 3: Финальный commit

```bash
git add -A
git commit -m "chore: final integration and cleanup"
```
