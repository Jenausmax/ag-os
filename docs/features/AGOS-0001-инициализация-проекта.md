## Описание

Создание скелета проекта AG-OS. Структура директорий для всех модулей, pyproject.toml с зависимостями, requirements.txt, базовая конфигурация config.yaml и заглушка main.py с argparse для выбора режима запуска (bot/tui/all).

## Acceptance Criteria

- [ ] Структура директорий создана (core, runtime, telegram, tui, scheduler, memory, guard, db, tests)
- [ ] pyproject.toml содержит все зависимости и dev-зависимости
- [ ] `pip install -e ".[dev]"` выполняется без ошибок
- [ ] `python main.py` выводит `AG-OS starting in all mode...`
- [ ] config.yaml содержит все секции конфигурации

## Затрагиваемые модули

- корень проекта: pyproject.toml, requirements.txt, main.py, config.yaml
- все модули: __init__.py файлы

## Ключевые интерфейсы

```python
# main.py
def main():
    parser = argparse.ArgumentParser(description="AG-OS: Multi-agent orchestrator")
    parser.add_argument("mode", choices=["bot", "tui", "all"], default="all", nargs="?")
    args = parser.parse_args()
```

## Edge Cases

- Отсутствие Python 3.11+ — pyproject.toml указывает requires-python >= 3.11
- Конфликты зависимостей при установке

## План реализации

### Step 1: Создать структуру проекта

```bash
mkdir -p ag-os/{core,runtime,telegram,tui,scheduler,memory,guard,db,tests}
touch ag-os/{core,runtime,telegram,tui,scheduler,memory,guard,db,tests}/__init__.py
```


### Step 4: Создать config.yaml

```yaml
telegram:
  token: ""  # BotFather token
  allowed_users: []  # Whitelist user IDs

agents:
  session_name: "ag-os"
  master:
    name: "master"
    model: "claude-cli"
    runtime: "host"
    type: "permanent"
  permanent: []

docker:
  defaults:
    cpus: 2
    memory: "4g"
    network: "bridge"
    workspace_base: "/data/ag-os/workspaces"
    shared_dir: "/data/ag-os/shared"

guard:
  enabled: true
  llm_enabled: true
  haiku_api_key: ""

scheduler:
  enabled: true

database:
  path: "ag-os.db"
```

### Step 5: Создать заглушку main.py

```python
import argparse
import asyncio


def main():
    parser = argparse.ArgumentParser(description="AG-OS: Multi-agent orchestrator")
    parser.add_argument(
        "mode",
        choices=["bot", "tui", "all"],
        default="all",
        nargs="?",
        help="Run mode: bot (Telegram only), tui (dashboard only), all (both)",
    )
    args = parser.parse_args()
    print(f"AG-OS starting in {args.mode} mode...")


if __name__ == "__main__":
    main()
```

### Step 6: Установить зависимости и проверить запуск

```bash
cd ag-os
pip install -e ".[dev]"
python main.py
```

Expected: `AG-OS starting in all mode...`
