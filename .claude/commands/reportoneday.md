Создать отчет о времени отработки команды Green team

**Действия**
1. Ты ДОЛЖЕН использовать MCP сервер mcp-atlassian
2. Спроси у пользователя на какое число нужен отчет
3. Нужно найти за [data] все записи времени участников грин тим, даже работы тестировщиков. Даже если они записывали время не в свои задачи
4. Записать все в файл отчета `doc/reports/xxxx-xx-xx-green-team-отчет-работ.md`
Пример названия файла: 2026-03-23-green-team-отчет-работ.md

Нужно заполнить заголовки:
# Green Team — Отчёт работ за [date]

### Разработка
### QA / Тестирование

## Итого за [date]

## По участникам

## Кто НЕ записал время [date]

## Задачи дня — детали

#### [ключ-таски] — Back:
